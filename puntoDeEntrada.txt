PRINCIPAL, tenes dos opciones:
1- Te pueden dejar crear tu proyecto desde cero con vite (npm create vite@latest) y te dejan utilizar React.
2- Te pueden dejar crear tu proyecto desde cero con vite (npm create vite@latest) y no te dejan utilizar react.
      Es decir, no te dejan inicializar un template hecho con react.
      Puedes instalar lo mínimo de vite y tu lo configures, es decir inicializar tu mismo el proyecto.

ENTONCES LO VAMOS A INICIALIZAR DE CERO CON VANILLA, ponemos el punto de entrada, VAMOS A CREAR EL PUNTO DE ENTRADA:
  EN LA CONSOLA PONEMOS LO SIGUIENTE:
    - npm create vite@latest
    - ponemos el nombre de la carpeta "pepito"
    - select a framework: vanilla
    - cd "pepito"
    - npm install @vitejs/plugin-react -E
    - YA TENEMOS EL PROYECTO CREADO Y ENTRAMOS A LA CARPETA:

Vamos a package.json y no tenemos la dependencia de React, hay que agregarla. Hay que instalar dos dependencias:
  - consola: npm install react react-dom -E
  - ya se agrego a las dependencias

Luego en la carpeta principal donde están todos los archivos creamos nuestra configuración de vite:
  - creamos un archivo llamado: vite.config.js
    - editamos el archivo recién creado:
        import { defineConfig } from "vite"
        import react from '@vitejs/plugin-react'

        export default defineConfig({
          plugins: [react()] 
        })

ELLOS NO LES INTERESA ESTO, LES INTERESA QUE SEPAS HACER UN PUNTO DE ENTRADA:
Cuando iniciamos nuestro proyecto en vite tenemos un index.html y un main.js
  en el index.html yo tengo un script que es el que estoy cargando al principio en mi pagina web que es el main.js
  y el main.js es el punto de entrada de mi aplicacion.
Y ELLOS TE VAN A DECIR, vale, como es el punto de entrada de tu app, como inicias una app con react?
  le respondemos como el main.js es el primer archivo que voy a cargar en mi app lo primero que voy a hacer es lo siguiente:
    borramos todo lo que hay adentro del archivo y tipeamos el siguiente codigo:
      import {createRoot} from 'react-dom/client'

      const root = createRoot(document.getElementById('app'))     ===> aca es donde quiero renderizar mi app de react, en id='app' que esta en index.html
      root.render(<h1>Hello, World</h1>)
SI AHORA HACEMOS EN LA CONSOLA: npm run dev, va a salir un eror.
  TENEMOS QUE CAMBIARLE DE MAIN.JS a MAIN.JSX
  si no cambiamos no va a poder ser capaz de transpilar el codigo, no esta preparado el plugin de vite que hemos instalado
  TAMBIEN CAMBIAMOS dentro de index.html el main.js a main.jsx
    AHORA SI YA TIENE QUE FUNCIONAR
- npm run dev
